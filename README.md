# Pygame_clock

Данный проект позволяет вывести системное время в отдельном окне.
Проект легко кастомизируется, можно изменить:
- Размер часов
- Цвет пикселей
- Смену пикселей на кружки
- Формат выводимого времени:
	+ 12/24 часовой вариант
	+ С секундами и без 

## Установка
``` shell
pip install pygame
```
